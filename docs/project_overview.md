# Project Overview

This project implements a **Face Emotion Detection system** using deep learning and computer vision.
The system detects faces in real time using OpenCV DNN and classifies facial emotions using a
CNN-based model trained with TensorFlow/Keras.

## Objectives
- Detect human faces from images or webcam streams
- Classify emotions into predefined categories
- Achieve stable real-time predictions using temporal smoothing

## Emotion Classes
- Angry
- Disgust
- Fear
- Happy
- Sad
- Surprise
- Neutral

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- MobileNetV2
- NumPy, scikit-learn

## Key Features
- Automatic handling of class imbalance
- Transfer learning with fine-tuning
- Real-time webcam inference
- Modular and scalable project structure
