# MobileNetV2 Details

MobileNetV2 uses:
- Depthwise separable convolutions
- Inverted residual blocks
- Linear bottlenecks

## Why MobileNetV2?
- High accuracy-to-parameter ratio
- Ideal for real-time inference
- Proven performance in vision tasks

## Transfer Learning
Weights are initialized from ImageNet and fine-tuned on the emotion dataset.
